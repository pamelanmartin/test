All tutorials should meet **WCAG 2.1 Level AA** — the federal standard required for higher education. Key requirements for every tutorial:

- Alt text on all images
- Heading hierarchy: one `<h1>` per page, logical `<h2>` / `<h3>` structure
- Color contrast ratio of at least **4.5:1** for body text
- All interactive elements (buttons, quizzes, drag-and-drop) must be usable by keyboard alone
- Visible focus outlines for keyboard users
- Drag-and-drop activities must have a keyboard-accessible alternative
- Never use color as the *only* way to convey meaning (e.g., correct/incorrect)
